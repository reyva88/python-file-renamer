# Python Automation Tools

This repository contains simple Python automation scripts designed to save time and automate repetitive tasks.

---

## 1️⃣ File Renamer

A script that automatically renames multiple files inside a folder using a custom prefix.

### Features
- Keeps original file extensions
- Renames files in bulk
- Easy to use via terminal input

---

## 2️⃣ CSV Cleaner

A script that cleans messy CSV files.

### Features
- Removes empty rows
- Removes duplicate rows
- Saves a cleaned version of the file

---

## Technologies Used
- Python 3
- Built-in libraries (os, csv)
