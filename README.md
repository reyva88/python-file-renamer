Python Automation Tools

This repository contains beginner-friendly Python automation scripts designed to save time and eliminate repetitive manual work.

These tools focus on practical, real-world problems that businesses and individuals frequently face.

📁 1. File Renamer

A Python script that automatically renames multiple files inside a selected folder using a custom prefix.

What It Does

Renames files in bulk

Preserves original file extensions

Helps organize large collections of documents, images, or reports

Example Use Case

Renaming hundreds of files like:
report1.pdf, report2.pdf, report3.pdf
into:
client_report_1.pdf, client_report_2.pdf, client_report_3.pdf

📊 2. CSV Cleaner

A Python script that cleans messy CSV datasets.

What It Does

Removes empty rows

Removes duplicate entries

Outputs a cleaned version of the original file

Example Use Case

Cleaning exported Excel data before analysis or reporting.

🛠 Technologies Used

Python 3

Built-in libraries (os, csv)

This project is part of my journey into automation and problem-solving with Python.

My focus is building simple, useful tools that save time and reduce manual work.
